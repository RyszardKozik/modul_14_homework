API Reference
=============

This section provides a detailed reference for all available API endpoints.

.. http:section:: Users API
.. http:post:: /api/users/

   Create a new user.

   **Request Body**:

   - **name** (str): User's name
   - **email** (str): User's email

   **Responses**:

   - **200**: User created successfully.
   - **400**: Bad request.
